# Lean

FormulationBench ships machine-checkable Lean 4 definitions for MILP
*formulations* and *reformulations*, alongside the Python loader. Every
formulation in the dataset is encoded as a `MILPFormulation` and every
reformulation as a `MILPReformulation`, both defined in `Common.lean`
at the dataset root.
